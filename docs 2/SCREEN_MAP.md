# Screen Map (All Screens + Ownership)

## User App (`apps/user-app`)
1) Login
2) Home / Choose Section (4 sessions)
3) Sales Entry (matrix)
4) Cart / Preview (if separate)
5) Pending Uploads (offline queue)
6) Edit/Delete Sales
7) Reports:
   - Number-wise
   - Net Pay
   - Winning
8) Result View (published results)

## Admin App (`apps/admin-app`)
1) Login
2) Dashboard
3) Sections master (sessions + series_config)
4) Schemes master (rates/payout/commissions)
5) Sales Group master (dropdown 1)
6) Sales Sub-Group / Book master (dropdown 2)
7) Ticket Books / Assignments
8) Blocked Numbers / Block Rules
9) Credit Limits
10) Users + Roles
11) Result Publish + Revoke
12) Exports (if required)

For navigation, see `NAVIGATION_MAP.md`.
