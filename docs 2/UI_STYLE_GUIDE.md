# UI Style Guide (APK-Faithful)

## Core colors (locked)
- Header / Primary: `#AA292E`
- Accent / Orange: `#F19826`
- Error / Timer red: `#C61111`
- Background: `#FFFFFF`

## Typography
- Prefer system font (Android)
- Headings: bold, high contrast
- Table text: compact, readable

## Components (pattern)
- Top header with title + countdown
- Stats strip (COUNT / Rs / TIME)
- Tab strip: 1 / 2 / 3 digits
- Dropdown row: Group + Book (two selectors)
- Numeric keypad and quick toggles

## Spacing
- Keep compact density; avoid modern “airy” redesign.
